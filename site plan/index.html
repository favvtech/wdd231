ModernEdge Interiors — Site Plan

Project planning document outlining naming, purpose, audience scenarios, color schema, typography, and wireframes.

1. Site Name

Name: ModernEdge Interiors

Reason: The name combines “Modern” and “Edge” to reflect a contemporary, Lagos-forward design identity—clean lines, bold accents, and functional luxury suitable for urban homes and workspaces.

Domain concept (example): modernedgeinteriors.com (subject to availability)

2. Site Purpose

The website presents ModernEdge’s interior design services, showcases projects, and enables prospective clients to reach out easily. Users can explore services and packages, browse projects with dynamic imagery, learn about our approach, and contact us with Lagos location mapping.

  • Services and packages overview with clear CTAs
  • Projects page with search/filter and dynamic images
  • About page that communicates story, philosophy, and credibility
  • Contact page with a real-time Lagos map and contact form

3. Scenarios

  • “What budget range should I expect for a living room redesign in Lagos?”
  • “Can I see examples of modern kitchens you’ve completed in Lekki or Ikoyi?”
  • “How long does a typical project take from consultation to completion?”
  • “Do you provide space planning and furniture selection for home offices?”

4. Color Schema

We’ll use a modern palette aligned with the brand and main site:

Primary: #e67e22
Use: buttons, accents, emphasis highlights
Text: #333333
Use: headings, body text, icons
Background: #FAFAFA
Use: page background, section fills

This document is styled using the schema above.

5. Typography

Primary Font: Inter

  • Headings (H1–H3): Inter 800/600 (bold, high contrast)
  • Body: Inter 400 for readability
  • Notes/Meta: Inter 400/500 with subtle color
